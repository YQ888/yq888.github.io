文件名：需要带有后缀

touch –help：查看能使用的参数选项。

创建名字是love的txt文本文件

创建多个文本文件，可以使用对象的方式

如果love.txt已经存在，再次创建则改变文件的时间戳

查看文件的状态：

其实mkdir也可以创建文件，但是一般创建文件还是使用touch，mkdir一般用来创建目录（文件夹）

vi和vim编辑器

如果vim或者vi提示无法使用。执行yum install vim即可

vi：

  • 只能编辑文本内容，不能对字体，段落进行操作
  • 不支持鼠标操作
  • 没有菜单，没有命令

vim：

  • 是vi的升级版本
  • 代码补全，支持鼠标操作，支持命令

vi和vim三种模式

  1. read：只读模式
  2. insert——i：插入模式
  3. 保存：wq

打开和新建文件

语法：vim 文件名字

  • 如果文件已经存在，则直接操作改文件
  • 如果文件不存在，则保存后新建一个文件。

vim的基本命令

编辑模式：

直接按键盘

  • i（insert）：编辑模式，在光标前面插入信息
  • l（insert）：编辑模式，在首行插入信息
  • a（append）：在光标后插入信息
  • A（append）：在末行插入信息
  • o：在当前光标插入一个空行
  • O：在当前行的前面插入一个空行

命令模式：

按键盘的esc，退出编辑模式，进入到命令模式

常见的保存命令：

  • w：保存
  • q：退出
  • q！：强制退出
  • wq：保存并退出
  • wq！：强制保存并退出
  • set number：显示行号

vim定位

查看文件，并定位到具体的行数

语法：vim 文件名字 + 行数

异常处理

vim异常退出，可能会出现交换文件

后缀是.swp的文件，在使用vim操作文件的时候，实际上会产生一个和操作文件一样的swp文件。等正常操作后，在对源文件进行覆盖。

删除swp文件即可

文件查看

  • cat 文件名：查看小文件内容
  • less -N 文件名：分屏查看大文件内容
  • head -n 文件名：查看文件的前一部分
  • tail -n 文件名：查看文件的最后一部分。
  • grep 关键字 文件名：通过关键字搜索文本文件的内容
  • more 文件名：查看大文件内容

cat，more，less的区别：

  1. more比cat强大，提供分页显示的功能
  2. less比more强大，提供翻译，跳转，查找等命令
  3. more和less都支持用空格显示下一页，按b显示上一页

cat

语法：cat （参数） 文件名

cat -n 文件名：查看，并显示行号 n（number)

less

语法：less（参数） 文件名

less-N 文件名：查看，并显示行号 n（number)

全屏操作

  1. ctrl + f：向前滚动一个屏幕
  2. ctrl +b：向后滚动一个屏幕
  3. ctrl +d：向前滚动半个屏幕
  4. ctr +u：向后滚动半个屏幕

单行操作

  1. j：往前滚动一行
  2. k：往后滚动一行

其他操作（区分大小写）

  1. G：移动到最后一行
  2. g：移动到第一行
  3. q/ZZ:退出less

tail

语法：tail（参数） 文件名

参数：

  • -数字：显示文件的最后几行 –3行

  • -f：循环读取文档的最后10行 –3行

  • -n<行数>：显示文件的内内容，从指定的位置到文件的末尾，从第四行开始

  • -c：显示最后指定的字节数

head

查看文件的前一部分,显示头文件n行内容

语法：head（参数） 文件名

-n :指定显示多少行

显示aa.txt的前10行

grep

进行查找

语法：grep（参数） 关键字 文件名

参数选项：

  • -n：查找包含关键字的行
  • -i：忽略大小写，查找包含关键字的行
  • -v：查找不包含关键字的行
  • -c:查找个数

查找sshd的进程

echo

echo会在终端显示参数指定的文字，通常会和重定向结合使用linux运行将命令执行结果重定向到一个文件将本该显示的终端上的内容输入或者追加到指定文件中

‘>’：表示输出，会覆盖文件原有的内容

‘>>’：表示追加，会将内容追加（新增）到文件的末尾

语法：echo 字符串

  • echo 字符串 > 文件名，将字符串写到文件中，覆盖文件内容
  • echo 字符串 >> 文件名，将字符串写到文件中，不覆盖文件内容，追加在文件末尾
  • cat 不存在的目录 &>> error.log:将命令失败的结果输入到error.log文件的末尾

awk

参考：Linux awk 命令 | 菜鸟教程 (runoob.com)

awk：处理文本文件的语言，是一个强大的文本分析工具

语法：awk（参数） ‘awk语法’ 文件

在love.txt文件中只看包含有y和z的行

/y|z/ 不能有空格，是严格检索。

  • -F ‘,’:按照逗号切割
  • $+数字：获取第几段内容
  • $0:获取当前行的内容。
  • OFS=”字符串”：向外输出分割的字符串
