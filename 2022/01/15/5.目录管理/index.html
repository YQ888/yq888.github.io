目录常用的命令

  • ls：列出列表
  • ll：相当于ls -l;
  • cd：切换目录
  • pwd：显示当前目录
  • touch：创建文件
  • mkdir：创建新的 目录
  • rmdir：删除空的目录
  • cp：复制文件或目录
  • rm：删除文件或者目录
  • mv：移动文件目录，或者修改文件或者目录的名字
  • cd .. ：回退上级目录

ls

ls：列出列表

ls –help:查看可用参数选项

ls -a:隐藏文件，.开头

ls -l:文件的详细信息

ls -al:显示所有文件和文件夹的详细信息

  • 第一列：权限，d开头表示目录，-开头表示文件，|开头表示链接文档
  • 第二列：文件的硬连接数或目录所含有的子目录数，即包括两部分的内容，第一部分是对文件，第二部分是对目录
  • 第三列：属于用户
  • 第四列：属于用户组
  • 第四列：文件大小

相关权限：

  • r：可读
  • w：可以写
  • x：可以执行
  • -：没有权限

权限按照-区分，第一部分，是属于用户的权限，第二部分是用户组的权限，第三部分是其他用户的权限。

ll(相当于ls -l)

是Long Listing format的缩写，列出当前位置文件及文件夹的明细列表，包括隐藏文件

pwd

显示当前文件的所在目录

pwd -P 和pwd效果一样

cd

cd (相对路径和绝对路径)

相对路径：相对于当前目录

绝对路径：从根目录开始

  • cd ~:切换到工作目录
  • cd .. :切换到上一级目录【.表示当前目录，..表示上一级目录】
  • cd ../..:切换到当前目录的上两层目录
  • cd -：切换到上一次所在的目录
  • cd /:切换到系统根目录
  • cd /usr:切换到usr目录

cd ..:返回到上级目录

mkdir

是make directory的缩写，创建目录（文件夹）

语法：mkdir (选项) 文件夹名字

在centos的可视化界面中查看

mkdir -p hello/word :创建多级目录，创建父级目录不存在的文件夹，递归创建目录。（创建多级目录）

系统中加粗的就是文件夹，没有后缀；带有后缀的，未加粗的就是文件

rmdir

删除空的目录

语法：rmdir 目录名字

rmdir -p 目录名字：删除多级目录

先删除word，如果删除了word后，hello也是空的，则删除hello

touch

创建文件

语法：touch 文件名

rm

删除命令

使用rm命令删除的文件不能恢复

语法：rm (选项) 文件或者目录

rm –help：查看更多选项

rm -r 目录名字：删除目录,recursive递归删除，删除目录及目录下所有文件，非空也可以删除

rm -f:force强制删除

rm -rf:强制删除任何文件

rm -rf *：强制删除当前目录下的所有文件

rm world*：删除以world开头的文件

rm *.txt：删除以.txt后缀的文件

这里注意，如果在当前文件夹目录下。使用rm -rf用绝对路径删除当前文件夹（所在目录）的时候，会删除当前目录上一级里所有的文件，也就是当前所在目录相对的所有文件都会被删除

cp

是copy的缩写，复制文件的命令

语法：cp（选项） 数据源 目的地

cp –help:查看选项参数

数据源的位置可以是相对路径也可以是绝对路径，这里的数据源只能是文件

cp -r 数据源/* 目标位置：拷贝文件夹所有文件到指定的位置，recursive递归赋值，如果给出的源文件是一个目录文件，此时将复制该目录下所有的子目录和文件。例如：cp -ai ./dir1./dir2

将demo中的所有文件，拷贝到hello目录下

  • -a：选项all通常在复制目录的时候使用，它保留链接、文件属性、并复制目录下的所有内容
  • -f：force强制覆盖，覆盖已经存在的目标文件而不给出提示
  • -i：-i和-f相反，在覆盖目标文件之前给出提示，要求用户是否确认覆盖，回答y才会执行覆盖

mv

移动文件或者文件夹，改数据名字，mv是move的缩写，剪切文件或者将文件改名

语法：mv （选项） 数据 目的地

mv –help:查看可以选择的选项

将demo目录下的所有数据移动到hello目录下

  • mv 数据文件名 目标文件名：实现改名
  • mv 数据文件 目标目录：将文件移动到目录
  • mv 数据目录 目标目录：目录已经存在，则将数据目录移动到目标目录下，目录不存在则修改数据目录名称
  • mv 数据目录，文件：报错

修改用户组和属性组

需要在root权限进行

chgrp

语法chgrp（参数） （所属群组）（文件或者目录）

修改yxqz文件的属性组

-v：带有提示

chown

  • 更改用户主，
  • 更改用户组的属性组

语法：chown 所属群组 文件名字

-R：指定目录以及子目录下的所有文件

修改demo的所有者为root

修改hello目录的所有者和属性组都是root

修改hello目录下的所有文件的所有者和属性组都是root

修改权限

chmod：修改所属用户主，所属用户组，其他用户权限

查看可以使用的参数选项：chmod –help

常用的依旧是-R：对当前目录下的以及里面的所有数据进行相同的设置

修改方式：数字和符号

数字方式

  • r（4）：可读
  • w（2）：可以写
  • x（1）：可以执行
  • -（0）：没有权限

rwx:4+2+1=7

不能使用重复的数字进行运算：比如6=4+2则是rw，不能是2+2+2即www

语法：chmod （参数） 数字权限 目标文件或者目录

设置demo目录中的所有文件，所属者可读可写可执行，用户组可读可写可执行，其他的没有权限操作

符号方式

  • usr ：属于者的权限，简写u
  • group：用户组的权限，简写g
  • others：其他用户的权限，简写o

符号权限：

  • +：添加权限
  • -：去掉权限
  • =：设置权限

设置hello目录中的所有文件，所属者可读，用户组可读，其他的保持默认权限

不设置，则就是保持默认，比如这里的其他o，没有设置权限，那么还是和之前的权限一样。

设置hello目录中的所有文件的权限全部设置成可读可写和执行

a：all的意思

去掉hello目录中的所有文件的其他用户可读可写可执行权限
